## ホロライブ非公式WEBサイト

理由
	－興味以外は何もないが、配信者たちの面白い場面をみんなと共有できるサイトがあれば面白いのでは
	ないかという目的で作っている
	
使用ツール

	- spring STS 3(3.15ver)
	- SQL(Oracle database)
	- bootstrap
	- js / ajax / jquery
	- API(Twitter / YouTube API) // 使用予定であって、まだ使うかどうかは作りながら決める
	- WEB運営のための aws 適用 or それに準する何か

	
使用する機能（機能以外にも開発したいところ）
	- CRUD 機能
	- 実時間情報のためのTwitterAPI
	- Youtube 映像を見せるときに使うAPI機能
	- メール機能　もしくはチャット機能を搭載（せめてチャットは入れて置きたい）
	- 会員登録機能と活動かつ管理者などの人それぞれに権限付与
	（会員登録の際、名前、ＩＤ，ＰＷ，ニックネーム、住所、Emailなど、必要な情報を取り扱う。）
	- その他、必要な機能はその時に記入して使うことにする
	
	
ロールモデル（*サイトを作るために参考にしたサイト）
	- ホロライブ非公式ＪＰ（https://seesaawiki.jp/hololivetv/）
